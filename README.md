# android-movieapp

# Goals
* Stack
    * Kotlin
    * Kotlin Coroutines
    * Retrofit
    * Concat Adapter
    * Nested Recyclerview
    * Navigation Components
    * LiveData
    * ViewModel
    * Room Persistance Library
    * Clean + MVVM Architecture
    * Single Activity Architecture
    * Bottom Navigation
    * Deeplink Navigation
    * Safe-args in Navigation
    * Multi-Module Project (Modularization)
    * Multi-Module Navigation
* Unit Test
    * MockK
    * Junit5 (Android Junit5)
