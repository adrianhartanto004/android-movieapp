# android-movieapp

# Goals
* Stack
    * Kotlin
    * Kotlin Coroutines
    * Retrofit
    * Navigation Components
    * LiveData
    * ViewModel
    * Room Persistance Library
    * Clean + MVVM Architecture
    * Single activity architecture + Navigation Components
    * Bottom Navigation
    * Deeplink navigation
* Unit Test
    * MockK
    * Junit5 (Android Junit5)
